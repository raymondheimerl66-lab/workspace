# Projects — Mission Control

## Produkt
Plattform für selbstständige Therapeuten in der Schweiz (Psychotherapie, Physiotherapie, Ernährungsberatung).

### Kernfunktionen
1) Terminbuchung (cal.com, self-hosted in CH)
2) Lokale Sprachaufnahme → KI-Zusammenfassung (Daten bleiben in CH, DSG/DSGVO)
3) Klientenverwaltung (digitale Patientenakte)
4) Abrechnung/Administration

## Ziel
Therapeuten sparen 30–50% Administrationszeit.

## Meilensteine (Vorschlag)
- M1 — Problem/Solution Fit & Compliance Baseline
- M2 — MVP Buchung + Patientenakte (ohne KI)
- M3 — Audio→Summary On-Device/Local + Abrechnung v1
